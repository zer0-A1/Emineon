Installation Instructions

  1. Download the manifest file using the button above
  2. Open Outlook (web, desktop, or mobile)
  3. Go to Settings → Add-ins → Get add-ins
  4. Click "Add a custom add-in" → "Add from file"
  5. Select the downloaded emineon-outlook-addin-manifest.xml file
  6. Click Install and accept the permissions
